## Captured Product/UX Decisions

- Preserve the existing SupportPRO logo unchanged.
- Preserve existing content meaning, service offerings, navigation links, testimonials, certifications, supported platforms, contact details, and footer links.
- Position SupportPRO as a trusted 24/7 outsourced technical support and infrastructure operations partner for hosting companies, MSPs, data centers, SaaS companies, and cloud businesses.
- Use a modern B2B SaaS plus cloud infrastructure visual style without making the site feel like a generic SaaS dashboard.
- Use a deep navy / blue / white / soft gray design language with limited cyan and urgent orange accents.
- Make Request for Quote the primary conversion action.
- Make Free Server Checkup the secondary conversion action.
- Make Emergency Support visually distinct and easy to reach.
- Replace crowded navigation with grouped mega menus on desktop and accordions on mobile.
- Replace slider-like hero behavior with a trust-first hero. Original dashboard concept superseded on 2026-05-29 by the warm support visual with floating cards.
- Move certifications and trust proof near the top of the homepage.
- Add a guided "What Type of Support Do You Need?" section without changing existing functionality.
- Group supported platform logos into capability categories.
- Show testimonials directly on desktop instead of relying only on a carousel.
- Modernize contact into a consultation-oriented lead generation section.
